Compatibility Key

Perfect: Game is fully emulated and all content can be played.

Great: Game runs and is almost completely playable, but may have some minor issues or missing some minor components.

Issues: Game runs and can play most content, but some components may not be emulated fully.

Unplayable: Game loads but may have some significant components not fully emulated.

Unknown: Compatibility not yet reported.
