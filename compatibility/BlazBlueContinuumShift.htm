BlazBlue: Continuum Shift

About

Hardware: Taito Type X2

Game Info: BlazBlue: Continuum Shift is a 2009 fighting game developed by Arc System Works as an official sequel to BlazBlue: Calamity Trigger. As with the previous game, the game first came to arcades before both the Xbox 360 and PlayStation 3 versions as the game was currently released for the Taito Type X2 arcade system board, with a 16:9 ratio and 768p resolution.

Common Issues / Questions
    How can I set the game in Free Play mode? Go to Test Mode (bind the key in Controller Setup) > System Configuration > Coin Setting "Free Play" How can I change the language? Go to Test Mode (bind the key in Controller Setup) > Game Configuration > Language "Japanese/English/Chinese/Hangul" How can I tweak the game options? Go to Test Mode (bind the key in Controller Setup) > Game Configuration The game doesn't save any changes made within Test Mode! Some games may require that a drive with letter D exists. Plug an USB stick or use an app like EaseUS Partition Master to split your HDD and create a D: drive.
