About

Game Title:
Initial D: Arcade Stage 4 (Export)

Hardware:
SEGA Lindbergh Yellow

Game Info:
Initial D Arcade Stage 4 is a racing game developed by Sega. It is the sequel to Initial D Arcade Stage Ver.3. This version has been heavily updated compared to its predecessors. Unlike the older games in this series, which were revisions of the original title, Initial D Arcade Stage 4 has a completely new physics engine written for the Sega Lindbergh arcade system. This new system allows cars to actually slide, or drift, across the track when clearing corners; this is a feature which is not present in the older games. Initial D Arcade Stage 4 features animated cel-shaded characters interacting with each other, while the cars and environment use updated, high-definition graphics.

Setup

Game

  • Executable is "id4.elf" found in "<game root>\disk0\" folder.

AMD Fix

  • Download fix here.
  • Open fix archive and open "D4Exp" folder.
  • Extract contents into "<game root>\disk0\" folder, replacing existing files.
  • Run "ID4Exp_fix.bat".
  • Enable "General - EnableAmdFix" in GAME SETTINGS.

LAN Networking (2 Players)

LAN Networking is currently broken and network connectivity does not appear to establish. In Store battle mode will be greyed out and Network testing in the Test Menu fails.

Setting up for LAN does not affect the ability to run this game in standalone mode.

Setup

  1. Edit the Game Settings within TeknoParrotUI.
  2. Set the "Ip" network IP address to the "IPv4 Address" of the current cabinet (Run "cmd.exe /k ipconfig").
  3. Set both "Gateway" and "Dns1" network IP address to your "Default Gateway" address.
  4. Set "BroadcastIP" to broadcast address of your IP/Subnet (this will be x.x.x.255 for a /24 IP Address on the Subnet 255.255.255.0).
  5. Set "Cab1IP" and "Cab2IP" to the IP address of the cabinet that will represent Seat Number 1 and 2, respectively (see Test Menu below).
  6. Enable the Test Mode option and run Game.
  7. Enter Game Assignments.
  8. Set a unique Seat Number and save.
  9. Exit and disable the Test Mode slider and re-run Game.

Always allow Window Firewall to pass through communication of this game. This should auto pop-up upon first execution (Alt+Tab to view or exit game).
