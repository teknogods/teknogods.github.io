Akai Katana Shin for NESiCAxLive

About

Hardware: Taito NESiCAxLive

Game Info: Akai Katana is a bullet hell shooter video game developed by Cave that was released on August 20, 2010 in arcades. A video game console port, Akai Katana Shin, was released on the Xbox 360 on May 26, 2011 in Japan. It is the fourth horizontal shoot 'em up game from the company, the prior three being Progear, Deathsmiles, and Deathsmiles II.

Setup
  • Game executable is 'game.exe' found in main folder.
  • The dll file iDmacDrv32.dll (in the game folder) should have a CRC32 = BCB0E7FE.
  • Delete d3d9.dll if it exists.
Common Issues / Questions
  • If you find a iDmacDrv32.dll.bak in the game folder after you've failed to launch a NESiCAxLive game, it's because your iDmacDrv32.dll doesn't have the right CRC32.
